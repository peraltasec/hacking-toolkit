# CyberSec Toolkit – Ferramentas Essenciais

Este repositório contém guias, scripts e laboratórios para quem está estudando **cybersegurança**.  
O objetivo é fornecer aprendizado prático, exploração ética e organização para estudos.

## Estrutura do projeto

- /tools → Guias detalhados de ferramentas por área
- /scripts → Scripts de exemplo para automação
- /captures → Arquivos PCAP para prática
- /screenshots → Prints de análise e exemplos

⚠️  pratique apenas em ambientes autorizados.
