# Mobile Hacking Lab – Ambiente Seguro

Guia para criar laboratório de **segurança mobile**.

## Ferramentas

- Android Studio / Emulador
- Burp Suite / Proxy
- APKs de teste (somente autorizados)

## Mini-lab

1. Instale emulador Android.
2. Configure Burp Suite para interceptar tráfego.
3. Instale APK de teste e explore comunicação.
4. Documente resultados em `/screenshots`.

## Dicas Éticas

- Teste apenas apps autorizados.
